# ILI9341-STM32F072B #

## Overview ##

This is a demonstration of a ILI9341 LCD using SPI connected to a STM32F072B Discovery board. 

## Connections ##

| Board | LCD |
| ------------- | ------------- |
| PB5 | DISP_DC |
| PB6 | SPI_CS |
| PB15 | SPI_MOSI |
| PB14 | SPI_MISO |
| PC13 | PI_SCK |
| PB12 | DISP_RESET |
| 3V | VCC |
| 3V | LCD |
| GND | GND |
